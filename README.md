# 👶 BabyCare Management System

## 📌 Overview

BabyCare Management System is a web-based application designed to help parents manage and monitor their baby's health-related information in one place. The system allows users to maintain baby profiles, track vaccinations, record growth details, monitor feeding and sleep patterns, and generate health reports.

The goal of this project is to provide an easy-to-use platform for organizing important baby care records digitally.

---

## ✨ Features

### 🔐 User Authentication
- User registration and login
- Secure session management
- Logout functionality

### 👶 Baby Profile Management
- Add multiple baby profiles
- Update baby details
- Delete profiles
- Upload and manage profile pictures

### 💉 Vaccination Tracker
- Add vaccination records
- Track upcoming and completed vaccinations
- Maintain vaccination history

### 📈 Growth Tracker
- Record baby's height and weight
- Monitor growth progress
- View growth records

### 🍼 Feeding Tracker
- Maintain feeding schedules
- Track feeding history

### 😴 Sleep Tracker
- Record sleep patterns
- Monitor sleep duration

### 💊 Medicine Reminder
- Add medicine details
- Manage medicine schedules

### 📄 Health Report Generation
- Generate baby health reports in PDF format
- Download and maintain health records

### 📊 Dashboard
- Overview of baby health information
- Quick access to different modules
